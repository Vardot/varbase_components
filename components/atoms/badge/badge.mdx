## Slots

### Examples
**Example #1:** New post badge.
```
  {% include 'varbase_components:badge' with {
    html_tag: 'a',
    color: 'text-bg-primary',
    url: forum.new_url,
    content: forum.new_text
  } %}
```

**Example #2:** Pill badges with success content
```
  {% include 'varbase_components:badge' with {
    html_tag: 'span',
    color: 'text-bg-success'
    content: 'Success',
    utility_classes: ['rounded-pill']
  } %}
```
