## Examples:

Pass rows for the Hero Slider content type from an unformatted view display which uses the Hero Card for the view mode. 
```
{%
  include "varbase_components:heroslider" with {
    id: id,
    controls: true,
    indicators: true,
    interval: 6000,
    keyboard: true,
    pause: "hover",
    ride: "carousel",
    touch: true,
    wrap: true,
    items: rows[0]['#rows'] ?? [],
  }
%}
```
